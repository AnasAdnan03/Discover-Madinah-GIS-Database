\# Discover Madinah — Graduation GIS \& Database Project



\## Project Overview



Discover Madinah is a graduation project focused on building an interactive GIS web application that helps users explore attractions in Madinah, Saudi Arabia.



The project combines structured location data, database design, interactive mapping, search filters, place details, user reviews, and chatbot assistance to improve how visitors and users discover religious, cultural, entertainment, and restaurant locations across the city.



> Note: This repository presents the project as a case study. The source code is not included at this stage.

\## Project Goals



\- Help users discover attractions in Madinah through an interactive map.

\- Organize location data using a structured database design.

\- Provide search and filtering by place category.

\- Display useful place information such as descriptions, images, ratings, and working hours.

\- Support users with chatbot assistance for easier exploration.



\---



\## Key Features



\- Interactive GIS map for Madinah attractions

\- Categorized locations: religious, cultural, entertainment, and restaurants

\- Search and filtering functionality

\- Place details with images, ratings, and working hours

\- User reviews and feedback

\- Chatbot assistance

\- Admin-side support for managing project data
